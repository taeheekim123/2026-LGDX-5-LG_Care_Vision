## User primary 99장 model-based prelabel package 생성

요청/배경:


판단:

```text
prelabel 생성은 이미 filter 클래스를 학습한 모델이 필요하므로 현재 배포된 best.pt를 사용했다.
YOLOv12는 최종 재학습 후보로 사용할 수 있지만, 아직 filter로 학습되지 않은 yolo12n.pt 기본 가중치로 prelabel을 만들면 filter bbox 초안이 되지 않는다.
로컬 작업공간 확인 결과 yolo12n.pt/yolov12n.pt는 없고, ultralytics 버전은 8.4.67이다.
공식 Ultralytics 문서상 YOLO12 detection weights는 yolo12n.pt 등으로 제공된다.
```
