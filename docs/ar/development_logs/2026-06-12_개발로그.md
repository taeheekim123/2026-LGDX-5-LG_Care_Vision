평가 후 결정:

```text
별도 학습 기반 classifier는 현 단계에서 도입하지 않는다.
147건은 학습 데이터가 아니라 golden evaluation set으로 사용한다.
우선 rule 보정, RAG 공식근거 coverage 보강, LLM adapter의 비권한 추출 보조 적용을 먼저 진행한다.
LLM API는 최종 안전 판단 권한을 갖지 않고 symptom/product/procedure/clarification 후보 추출에만 사용한다.
```

남은 한계/후속:

```text
clarification gate가 긴 VOC 문장에 과도하게 질문을 요구한다.
remote/mode keyword가 세탁기/공기청정기 문맥에서 remote_operation으로 과매칭된다.
air purifier filter_replacement와 filter_cleaning 우선순위 보정이 필요하다.
non-AC 절차 공식 RAG coverage 부족으로 official_evidence_required false positive가 생긴다.
```
