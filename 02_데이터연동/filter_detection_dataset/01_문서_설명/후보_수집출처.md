## 2. 주의해야 하는 후보

| 후보 | 판단 |
|---|---|
| 유튜브 영상 캡처 | 개인 학습 실험은 가능할 수 있으나 발표/산출물/공유 데이터셋에는 저작권 리스크가 있다. |
| 블로그 이미지 | 이미지마다 권리자가 달라 일괄 사용이 어렵다. 사용 허가 또는 명시 라이선스가 없으면 제외한다. |
| iStock/Getty/Shutterstock 등 stock preview | 학습 데이터로 무단 수집하지 않는다. 필요한 경우 정식 라이선스 구매 후 사용한다. |
| 차량/산업용 필터 단독 이미지 | `filter` 일반 형태 학습에는 도움이 될 수 있으나, 에어컨 내부 필터 위치 탐지에는 오탐을 만들 수 있어 1차 학습에서는 제외한다. |
| 공식 매뉴얼 렌더링 페이지 | 공식근거/라벨 기준으로는 유용하지만, 실제 카메라 장면 학습 핵심으로 사용하면 도식/문서 패턴에 과적합될 수 있다. |

## 4. 1차 수집 결론

공개 데이터만으로 `에어컨 내부 필터 bbox` 100~200장을 바로 안정적으로 구성하기는 어렵다.

따라서 1차 POC는 다음 조합으로 간다.

```text
직접 촬영/허가 이미지 60~100장
+ 공개 라이선스 후보 30~60장
+ negative sample 20~40장
+ 로컬 reference seed 4장
```

이후 Roboflow에서 `filter` bbox만 라벨링하고, 학습 결과가 흔들리면 직접 촬영 이미지를 200장 이상으로 늘린다.
