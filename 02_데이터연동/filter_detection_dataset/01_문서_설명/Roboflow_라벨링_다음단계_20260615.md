## Roboflow 검수 규칙

```text
1. Project Type은 Object Detection이어야 한다.
2. class는 filter 하나만 사용한다.
3. bbox는 필터 그물망/필터 프레임이 실제로 보이는 영역만 감싼다.
4. 에어컨 전체, 커버 전체, 손, 벽면, 배경은 bbox에 포함하지 않는다.
5. 필터가 거의 보이지 않거나 판단 불가한 이미지는 rejected/review_needed로 제외한다.
6. pre-label bbox가 과하게 넓거나 화면 가장자리에 붙어 있으면 반드시 수정한다.
7. Generate Version 전에 train/valid/test에 review_needed 또는 rejected 이미지가 들어가지 않았는지 확인한다.
```
