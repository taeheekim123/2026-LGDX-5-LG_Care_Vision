## 3. 해석

기존 최종 선별 115장은 1번 데이터셋 구성의 broad 최소 수량 조건은 넘겼지만,
벽걸이 시연 기준에서는 창문형/도식 후보가 섞여 있어 core training seed로 쓰지 않는다.

벽걸이형 기준 현재 우선 후보는 사용자 제공 primary 99장이다.
기존 reviewed 86장을 함께 쓰는 extended 후보는 185장이다.

다만 LG 공식 지원 이미지는 도식/안내 이미지 비중이 높고, iFixit 실사진은 라이선스 확인이 필요하다.
YOLO가 실제 웹캠 장면에서 안정적으로 내부 필터를 찾으려면 직접 촬영 또는 명시 허가 사진을 추가하는 것이 여전히 좋다.

따라서 다음 보강 목표는 아래처럼 조정한다.

```text
사용자 제공 primary 99장 Roboflow 업로드 및 filter bbox 라벨링
필요 시 기존 reviewed 86장을 더해 extended 185장으로 2차 학습
negative sample 20~40장 추가
```

## 4. Wikimedia 시도 결과

Wikimedia Commons API에서 후보 검색은 가능했으나 이미지 파일 다운로드 단계에서 429 robot policy 응답이 발생했다. 원본 다운로드와 thumbnail 다운로드 모두 실패했다.

따라서 Wikimedia는 자동 수집 대상이 아니라 수동 선별 후보로만 남긴다.
