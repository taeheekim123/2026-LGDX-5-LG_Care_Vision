# 데이터연동 코드파일 한글 설명

작성일: 2026-06-04

## 1. `db/schema.sql`

SQLite DB 테이블 구조를 정의한다.

주요 테이블:

```text
users
devices
usage_logs
smart_diagnosis_results
environment_contexts
official_assets
official_document_chunks
official_document_embeddings
rag_search_logs
safety_audit_logs
product_models
part_maps
ar_guide_steps
chat_sessions
chat_messages
conversation_state
intent_risk_test_cases
intent_risk_eval_results
```

이번 작업에서 `official_document_embeddings` 테이블이 추가되었다.

## 2. `db/seed_ar_mock_db.py`

`mock_data/*.json` 파일을 SQLite DB인 `careshot_ar_mock.db`에 적재하는 seed script다.

주의: seed를 다시 실행하면 DB가 초기화된다. embedding vector는 seed 이후 `scripts/구축_공식문서_임베딩_VectorDB.py`를 다시 실행해 생성해야 한다.

## 3. `db/ar_db_repository.py`

백엔드, AI 판단엔진, RAGService가 DB를 조회할 때 사용하는 repository 계층이다.

주요 함수:

| 함수 | 의미 |
|---|---|
| `get_user_profile()` | 고객 프로필 조회 |
| `get_device_context()` | ThinQ 등록 제품 mock 조회 |
| `get_usage_log()` | 사용 로그 조회 |
| `get_smart_diagnosis()` | 스마트 진단 결과 조회 |
| `get_environment_context()` | 인도 지역 환경 context 조회 |
| `find_official_assets()` | 공식자료 strict matching |
| `search_official_document_chunks()` | lexical fallback용 chunk 검색 |
| `search_vector_official_document_chunks()` | embedding table 기반 vector 후보 검색 |
| `get_embedding_stats()` | embedding 구축 상태 확인 |
| `create_rag_search_log()` | RAG 검색 로그 저장 |

## 4. `scripts/build_evidence_based_data.py`

실제 VOC, 실제 환경 API, LG India 공식자료를 바탕으로 근거 기반 mock JSON을 생성하는 스크립트다.

## 5. `scripts/enhance_rag_official_corpus.py`

LG India 공식 PDF(Owner's Manual/Spec/Dimension 포함), Online Manual, Help Library, 제품 지원 페이지를 추가 수집하고 `official_assets_db.json`, `official_document_chunks.json`을 확장하는 스크립트다.

## 6. `scripts/구축_공식문서_임베딩_VectorDB.py`

6번 Embedding/Vector DB 구축용 스크립트다.

수행 내용:

```text
official_document_chunks 조회
→ careshot_local_hashing_v1 embedding 생성
→ official_document_embeddings 테이블 저장
→ vector_index/official_document_embeddings.jsonl 저장
→ official_document_chunks.embedding_status = embedded 갱신
→ 06_산출물/Embedding_VectorDB_검증리포트_2026-06-04.md 생성
```

현재 구축 결과:

```text
official_document_chunks: 1,890
official_document_embeddings: 1,890
embedding_status=embedded: 1,890
```

## 7. 공식 URL 기준

RAG와 embedding 대상 공식자료 도메인은 아래 두 개다.

```text
https://www.lg.com/in/
https://gscs-manual.lge.com/
```

`gscs-manual.lge.com`은 LG Online Manual 원본 도메인이므로 공식자료 범위에 포함한다.
